## ✍ Loop
Code that repeats **for** a certain number of times, or **while** something is true.

## ✍ `for` Loop
Code that repeats **for** a certain number of times.

```python
numbers = [1,2,3]
for x in numbers:
    print "This is number" + str(x)
```
    This is number 1
    This is number 2
    This is number 3
